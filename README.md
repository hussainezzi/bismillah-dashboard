# Bismillah Traders Dashboard 📊

Live sales & inventory dashboard for Bismillah Traders.

**Live URL:** https://hussainezzi.github.io/bismillah-dashboard/

## Data Refresh

Data is generated from Odoo XML-RPC and pushed to this repo periodically.

## Tech

- Pure HTML + Chart.js + Bootstrap 5
- Data fed via `data.json`
- Hosted on GitHub Pages
